### A. The "Smart Tooltip" Engine
Do NOT hardcode specific HTML for every tooltip. Instead, build a universal listener.

**HTML Usage (How to use it in templates):**
```html
<!-- Lawyer Persona (Blue Theme) -->
<button class="btn-upload" 
    data-michael-persona="lawyer" 
    data-michael-tip="Your files stay on your device. I never see them.">
    Upload PDF
</button>

<!-- Psychologist Persona (Green/Warm Theme) -->
<div class="loading-spinner" 
    data-michael-persona="psychologist" 
    data-michael-tip="Breathe in... Breathe out... formatting normally takes 3 seconds.">
</div>
```

**JavaScript Logic (pseudo-code):**
1. Listen for `mouseenter` on any element with `[data-michael-tip]`.
2. Dynamically create a DOM element (`.michael-whisper-bubble`).
3. Inject the correct **Avatar Image** based on `data-michael-persona`.
4. Inject the text from `data-michael-tip`.
5. Position it near the mouse/element with a bouncy CSS animation.
